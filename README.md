# Cyb3rClaw 🦞\n\nCyb3rClaw is a fork of [PicoClaw](https://github.com/sipeed/picoclaw) optimized for hybrid low-resource deployment on Raspberry Pi Zero + LM Studio inference on gaming PC.\n\n## Features\n- Pi Zero orchestration (armv6/7)\n- LM Studio proxy (Qwen2.5-Coder-3B validated)\n- Hooks for stats, budget, validation, context slicing\n- L1→L2→L3 agent hierarchy\n- Budget-aware execution\n\n## Upstream Pin\nSee [docs/upstream_pin.md](docs/upstream_pin.md)\n\n## Build\nmake build\n\n## Pi Zero\nGOOS=linux GOARCH=arm GOARM=6 make build (adjust Makefile if needed)\n\n## Config\nUse test_config.json for LM Studio validation.\n\n## Phases\nSee [EXECUTION_PLAN.md](../docs/EXECUTION_PLAN.md)
